## 📕️ Como baixar o projeto

```bash
    #Clonar o repositório
     git clone  https://github.com/Liliamsn/web-nlw8

    #Entrar no diretório 
     cd web-nlw8

    #Instalar o live-server globalmente
     npm install 

    #Inicializar o projeto
     npm run dev


     
```

 Acesse o projeto em http://localhost:3000/
